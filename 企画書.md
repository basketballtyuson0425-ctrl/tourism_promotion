# 伊勢観光分析 企画書

## 1. 企画概要

「伊勢観光分析」は、伊勢志摩に関する YouTube 上の旅行・観光コンテンツを分析し、海外旅行者に向けた観光プロモーションの改善に活用するためのプロトタイプである。

本企画では、伊勢志摩の観光資源が海外旅行者にどのように受け取られているかを把握し、今後の情報発信や施策立案につなげることを目的とする。

## 2. 背景

近年、日本全体では訪日外国人観光客の回復が進んでいる。京都、広島・宮島、富士山周辺など、SNS や動画で話題化しやすい地域では、海外からの関心が高まりやすい状況がある。

一方で、伊勢志摩には伊勢神宮、英虞湾、海産物などの豊富な観光資源があるにもかかわらず、海外向けの発信や SNS 上での広がりには改善余地があると考えた。

そこで、伊勢志摩を対象地域として選定し、旅行者目線の投稿を分析することで、より効果的な観光プロモーションにつなげる仕組みを検討した。

## 3. 対象地域の選定理由

伊勢志摩には、海外旅行者に訴求しやすい観光資源が複数存在する。

- 伊勢神宮: 日本文化を体験できる代表的な観光資源
- 英虞湾・リアス海岸: 自然景観、クルーズ、リゾート体験につながる観光資源
- 食文化: 伊勢海老、牡蠣、海鮮、松阪牛など、旅行目的になりやすい食の魅力

これらの魅力は十分にあるものの、海外旅行者が実際に何に関心を持ち、どのように発信しているかを継続的に把握する仕組みが必要だと考えた。

また、三重県全体のインバウンド回復は、全国と比較して遅れが見られる。三重県の「みえインバウンド誘客計画（仮称）中間案」では、三重県の外国人延べ宿泊者数について、2024年は24.1万人で、コロナ禍前である2019年からの回復率は62％、全国47位とされている。さらに、2025年速報値では37.2万人まで増加しているものの、回復率は96％、全国39位にとどまっている。

一方、JNTOの発表では、2025年の年間訪日外客数は42,683,600人となり、2024年を上回って過去最多を更新している。全国では訪日需要が大きく回復している一方で、三重県はインバウンド回復率や外国人宿泊者比率の面でまだ伸びしろがあるため、伊勢志摩を対象地域として選定した。

- JNTO「訪日外客数（2025年12月推計値）」: https://www.jnto.go.jp/news/press/20260121_monthly.html

## 4. 現状課題

伊勢志摩の観光プロモーションにおける課題は、以下の通りである。

- 自治体や公式サイトからの発信が中心になりやすい
- 単発の PR で終わりやすい
- 海外旅行者が何に反応しているか把握しにくい
- YouTube や SNS 上の投稿傾向を継続的に確認しにくい

特に、海外旅行者の視点で「何が魅力として伝わっているのか」を把握することが重要である。

## 5. 対象ターゲットの選定理由

本企画では、対象ターゲットを米国に設定する。

理由は、今回のYouTube検索語が英語中心であり、取得できる動画データと分析対象市場を一致させるためである。

米国は、訪日旅行における消費額が大きい市場であり、英語の旅行動画も多い。そのため、YouTube上の英語旅行投稿を分析する今回のプロトタイプでは、まず米国を優先ターゲットとして扱う。

現時点の検索語は英語が中心である。分析条件と画面表示をそろえるため、今回のデモでは対象市場を米国に絞る。

| 観点 | 今回の扱い |
|---|---|
| 対象市場 | 米国 |
| 分析対象言語 | 英語 |

## 6. 分析対象

本企画では、分析対象を YouTube に絞る。

YouTube を選定した理由は、旅行動画では訪問先、体験内容、食事、景色、移動の流れなどが具体的に表現されやすく、観光地の魅力や課題を読み取りやすいためである。

加えて、YouTube は公式の YouTube Data API が整備されており、動画タイトル、説明文、再生数、コメントなどを取得する設計を検討しやすい。Google Developers の公式資料では、YouTube Data API を有効化したプロジェクトには標準で1日10,000ユニットのクォータが割り当てられるとされている。今回のようなプロトタイプ段階では、API利用の見通しを立てやすい点も、YouTubeを優先する理由である。

一方で、TikTok、Instagram、X は、研究利用の審査、ビジネスアカウント要件、有料プランやレート制限などがあり、初期段階のプロトタイプでは実装難度が上がりやすい。

### SNS API利用しやすさの比較

| SNS | API利用のしやすさ | 主な取得対象 | 主な制限・注意点 | 今回の優先度 |
|---|---|---|---|---|
| YouTube | 高い | 動画タイトル、説明文、再生数、コメントなど | クォータ制限はあるが、公式APIが整備されており、プロトタイプで設計しやすい | 高 |
| TikTok | 低い | 投稿データ、研究用途のデータなど | Research API は申請・審査が必要で、承認まで時間がかかる可能性がある | 低 |
| Instagram | 中〜低 | ビジネス・クリエイターアカウントの投稿、コメント、インサイトなど | Instagram のプロアカウントやMeta側の権限設定が必要で、一般投稿の横断的な収集には向きにくい | 中〜低 |
| X | 中〜低 | 投稿、検索結果、ユーザー情報など | プランごとの利用制限やレート制限があり、安定した大量収集にはコスト・制限面の確認が必要 | 中〜低 |

参照URL:

- YouTube Data API Overview: https://developers.google.com/youtube/v3/getting-started
- TikTok Research API: https://developers.tiktok.com/products/research-api/
- Instagram Platform: https://developers.facebook.com/docs/instagram-platform/
- X API Rate Limits: https://docs.x.com/x-api/fundamentals/rate-limits

分析対象の言語は、対象ターゲットに合わせて英語に絞る。

| 対象市場 | 分析対象言語 |
|---|---|
| 米国 | 英語 |

将来的には Instagram や TikTok なども分析対象に追加できるが、今回のプロトタイプでは、まず説明しやすく実現可能な範囲として YouTube を中心に設計する。

## 7. 比較対象

比較対象は広島県（宮島）に絞る。

宮島を選定した理由は、伊勢志摩と観光資源の性質が近いためである。特に、神社を中心とした文化観光、海に面した景観、海鮮を含む食文化という点で共通している。

また、主要都市からの移動に一定の手間がかかる点も類似している。伊勢志摩は名古屋や大阪方面からの移動、宮島は広島市内から電車やフェリーを乗り継ぐ移動が必要であり、目的地までの行き方を分かりやすく伝えることが観光促進において重要となる。

| 地域 | 特徴 | 参考にする観点 |
|---|---|---|
| 広島県（宮島） | 神社、海景観、海鮮、海外認知 | 文化観光と海景観を組み合わせた見せ方、アクセス情報の伝え方、海外向け動画発信数 |
| 伊勢志摩 | 神社、海景観、海鮮・食文化 | 魅力は多いが、海外向けの動画発信数や見せ方に改善余地がある |

宮島をそのまま模倣するのではなく、伊勢志摩と近い観光資源を持つ地域が、海外旅行者にどのように伝わっているかを比較対象として参考にする。

## 8. 解決方針

本プロトタイプでは、次の3点を実現することを目指す。

1. 伊勢志摩に関する YouTube 投稿の傾向を整理する
2. 海外旅行者が反応しやすい観光テーマを把握する
3. 分析結果をもとに、次に取るべき情報発信案を提示する

単に投稿を集めるだけでなく、分析結果を観光プロモーションの改善案につなげる点を重視する。

## 9. システム概要

「伊勢観光分析」では、以下の流れを想定する。

1. 伊勢志摩に関する YouTube 動画や投稿情報を収集する
2. 英語の旅行投稿を中心に、タイトル、説明文、再生数、コメント、使用されている言葉を整理する
3. 人気のある観光テーマや、海外旅行者が反応している内容を確認する
4. 画面上で分かりやすく表示する
5. 観光プロモーションの改善案を提示する

今回の提出では、YouTube APIで取得した保存済み実データを用いて、画面と考え方を示す。

## 10. デモで示す内容

デモでは、完成したシステムではなく、課題へのアプローチと画面イメージを示す。

- 分析の全体像が分かるトップ画面
- 対象市場と言語を確認できる画面
- 人気キーワードを確認できる画面
- 文化、自然、食などの観光テーマ別の傾向を確認できる画面
- 広島県（宮島）との比較を確認できる画面
- 分析結果から発信アイデアを提示する画面

これにより、観光担当者が「どの魅力を、どのように発信すべきか」を考えるための補助ツールとして位置づける。

## 11. 工夫した点

本企画で工夫した点は、以下の通りである。

- 対象地域を伊勢志摩に絞り、課題設定を明確にした
- 対象ターゲットを、現在の英語検索語と一致する米国市場に絞った
- 分析対象言語を英語に絞り、見るべき投稿範囲を明確にした
- 分析対象を API利用の制限が少ないYouTube に絞り、実現可能な範囲にした
- 比較対象を広島県（宮島）に変更し、観光資源の類似性を踏まえて比較しやすい構成にした
- 観光資源を「文化・自然・食」に分け、整理しやすくした
- 分析結果を、単なるデータ表示ではなく施策案につなげる構成にした


- 技術の完成度だけでなく、課題の捉え方と改善の流れが伝わるようにした

## 12. 使用技術

提出用デモは、HTML・CSS・JavaScript と Node.js の最小構成で作成する。

- HTML: 画面構成
- CSS: デザイン、レイアウト
- JavaScript: 表示内容の切り替えやデータ表示
- Node.js: YouTube API接続、保存済みデータの読み出し、集計

複雑な仕組みを増やしすぎず、実装内容を説明できる範囲で構成する。

## 13. 今回の対象外

限られた期間で提出可能な形にするため、以下は今回の実装対象外とする。

- 本番運用を想定した大規模なAPI連携
- 定期実行による大量データの自動収集
- コメント本文を使った詳細な感情・意見分析
- AI による完全自動分析
- ログイン機能
- 本番運用を前提としたサーバー構築

これらは今後の拡張要素として位置づけ、今回の提出では企画意図と画面イメージを優先する。

## 14. 企画の要点

この企画では、以下の流れを重視する。

1. 伊勢志摩には観光資源が多い一方で、海外向け発信には改善余地がある
2. 現在の英語検索語に合わせ、米国を優先ターゲットにした
3. YouTube の英語旅行投稿を分析し、海外旅行者が何に関心を持っているかを把握する
4. 比較対象として広島県（宮島）を設定し、神社・海景観・海鮮の食文化・アクセス面の伝え方を参考にした
5. デモでは、対象市場、人気キーワード、観光テーマ、比較結果、発信アイデアまでを一連の流れで示す
6. 今回は完成品ではなく、課題の捉え方と解決までの流れを伝えるプロトタイプとして作成した

## 15. 今後の改善案

今後は、以下のような発展を想定している。

- コメント本文を取得し、海外旅行者の感想を分類する
- Instagram や TikTok も分析対象に追加する
- 季節ごとの関心の変化を確認できるようにする
- 定期実行によるデータ自動更新
- 観光担当者向けのレポート出力機能を追加する

まずは小さく作成し、分析対象や機能を段階的に広げていく方針である。
