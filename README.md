# 伊勢観光分析 tourism promotion mock

観光プロモーション支援ツール「伊勢観光分析」の画面モックです。

HTML・CSS・JavaScript の簡単な構成で、ブラウザから画面一覧を確認できます。

## 使い方

1. `04_src/frontend/index.html` をブラウザで開く
2. 提出用デモを確認する
3. 必要に応じて `伊勢観光分析_画面モック.html` で既存の画面モックを確認する

## 提出用デモ

- `04_src/frontend/index.html`: 提出用デモの入口
- `04_src/frontend/style.css`: デモ画面の見た目
- `04_src/frontend/script.js`: デモ画面のデータ表示とボタン操作

## 既存モックの確認

1. `伊勢観光分析_画面モック.html` をブラウザで開く
2. 画面モックを確認する
3. 必要に応じて `screens` フォルダ内の各画面を編集する

## 主なファイル

- `04_src/frontend/`: HTML・CSS・JavaScript の提出用デモ
- `伊勢観光分析_画面モック.html`: 既存の画面モックを表示する入口ファイル
- `design-canvas.jsx`: 複数画面を並べて見るためのキャンバス
- `theme.jsx`: 色、文字、共通部品などの設定
- `screens/`: 各画面の JSX ファイル
- `uploads/`: 画像などのアップロード用フォルダ

## 開発メモ

このプロジェクトは、まず画面の見た目や流れを確認するためのモックです。

大きな設計よりも、最初は「ブラウザで開いて画面を確認できること」を優先します。
