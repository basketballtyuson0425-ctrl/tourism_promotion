# YouTube API 検索条件

## 1. 目的

YouTube APIで実データを取得する前に、伊勢志摩と宮島でどの検索語を使うかを整理する。

検索語を先に決めておくことで、取得する動画のばらつきを減らし、伊勢志摩と宮島を同じ条件で比較しやすくする。

## 2. 検索対象

| 地域 | 目的 |
|---|---|
| 伊勢志摩 | 分析対象の中心地域 |
| 広島県（宮島） | 比較対象地域 |

## 3. 伊勢志摩の検索語

| No | 検索語 | 狙い |
|---:|---|---|
| 1 | `Ise Shima travel` | 海外旅行者向けの全体的な旅行動画を拾う |
| 2 | `Ise Jingu travel` | 伊勢神宮を中心にした文化観光動画を拾う |
| 3 | `Ise Grand Shrine travel` | 英語圏で使われやすい表記を拾う |
| 4 | `Ise Shima food` | 食文化、海鮮、食べ歩き動画を拾う |
| 5 | `Ama divers Ise Shima` | 海女文化に関する動画を拾う |
| 6 | `Ago Bay cruise` | 英虞湾、海景観、クルーズ系の動画を拾う |
| 7 | `伊勢志摩 観光` | 日本語動画も参考として拾う |
| 8 | `伊勢神宮 外国人 観光` | 訪日客視点の動画を拾う |

## 4. 宮島の検索語

| No | 検索語 | 狙い |
|---:|---|---|
| 1 | `Miyajima travel` | 宮島全体の旅行動画を拾う |
| 2 | `Itsukushima Shrine travel` | 厳島神社を中心にした文化観光動画を拾う |
| 3 | `Miyajima torii gate` | 海上鳥居、海景観の動画を拾う |
| 4 | `Miyajima food` | 牡蠣、あなごめしなど食文化動画を拾う |
| 5 | `Hiroshima Miyajima travel` | 広島市内から宮島への周遊動画を拾う |
| 6 | `Miyajima ferry` | アクセス、フェリー移動に関する動画を拾う |
| 7 | `宮島 観光` | 日本語動画も参考として拾う |
| 8 | `厳島神社 外国人 観光` | 訪日客視点の動画を拾う |

## 5. 取得時の基本条件

| 条件 | 方針 |
|---|---|
| 取得期間 | 直近1年を基本にする |
| 取得件数 | 1検索語あたり最大50件 |
| 重複動画 | `videoId` で重複を除外する |
| 並び順 | まずは関連度順、必要に応じて投稿日順も試す |
| 対象言語 | 中国語（簡体字・繁体字）、韓国語、英語、広東語表現を優先 |

## 6. 注意点

- YouTube検索には観光と関係ない動画も混ざるため、タイトル・説明文・タグで後から絞り込む。
- `search.list` はAPI利用量が重いため、検索語を増やしすぎない。
- 伊勢志摩と宮島で検索語の数と取得条件をそろえる。
- 最初は完璧な分析よりも、比較できる最低限のデータを取ることを優先する。
